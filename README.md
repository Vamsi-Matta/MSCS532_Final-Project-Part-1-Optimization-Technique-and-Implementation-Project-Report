# Data Structure Optimization using Hash-Based Lookup

## Project Overview
This project demonstrates how selecting the right data structure improves performance in high-performance computing (HPC) scenarios.

## Objective
To compare the efficiency of:
- List-based duplicate detection (O(n²))
- Set-based duplicate detection (O(n))

## Optimization Technique
Hash-based lookup using Python sets.

- main.py → Python implementation
- outputs/ → Screenshots of output and graph
- docs/ → Additional explanation
